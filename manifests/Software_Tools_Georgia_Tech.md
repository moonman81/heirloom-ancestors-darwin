# Software Tools (Georgia Tech port of Kernighan & Plauger's book)

The Georgia Tech Software Tools tape — Kernighan & Plauger's 1976
*Software Tools* implementations, ported to Ratfor and shipped
as a re-installable tape environment.  Historically important for
the "tools philosophy" that shaped subsequent Unix.

## Location on TUHS Archive

- `https://www.tuhs.org/Archive/Applications/Software_Tools/Georgia_Tech/`
  (~600 MB, includes multiple donor tapes)

## Content

- `c_tape/` — one of the donor tapes
- Original Ratfor sources for many tools that later became POSIX
  standards.

## Why this matters to Heirloom

Historically significant but not directly ancestral to Heirloom's
SVR4 line.  Complements the port bibliographically:
Kernighan-Plauger 1976 introduces the "small, sharp tools" ethos
that Heirloom preserves.
