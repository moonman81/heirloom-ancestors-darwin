# KSOS — Kernelized Secure OS (Ford Aerospace, ~1980)

Ford Aerospace's kernelised secure Unix — an early Multi-Level
Security (MLS) Unix designed to Bell-La Padula policy.
Historically significant for CIA/DoD secure-OS scholarship.

## Location on TUHS Archive

- `https://www.tuhs.org/Archive/Distributions/Other/KSOS/`

## Why this matters to Heirloom

Illustrates the security-hardened Unix lineage. No direct code
descendants in Heirloom but conceptually foundational for the
security posture heirloom-workspace-darwin's `hardening/` sweep
inherits.
